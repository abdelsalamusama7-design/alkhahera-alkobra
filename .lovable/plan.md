# خطة بناء موقع "القاهرة الكبرى" الإخباري

موقع إخباري عربي (RTL) احترافي على غرار MSN News واليوم السابع والأهرام، مع الحفاظ على هوية الموقع الحالي (الشعار وأسماء الإدارة).

## الهوية البصرية
- **الاسم:** القاهرة الكبرى (شعار بنفس الطابع: أهرامات + قلعة)
- **الاتجاه:** RTL كامل، خط عربي احترافي (Cairo / Tajawal / IBM Plex Sans Arabic)
- **الألوان:** كحلي عميق (#0B2447) كلون أساسي + ذهبي (#C9A227) للهايلايت + أبيض/رمادي فاتح للخلفيات + أحمر (#D62828) لشريط "عاجل"
- **الإحساس العام:** صحيفة رقمية مهنية، كثافة محتوى عالية، بطاقات نظيفة بظلال خفيفة

## هيكل الصفحات
- **الصفحة الرئيسية** (تركيز هذه الجولة)
- صفحة قسم (أخبار / رياضة / اقتصاد...) — هيكل قابل لإعادة الاستخدام
- صفحة تفاصيل خبر

## مكونات قابلة لإعادة الاستخدام
- `NewsCard` (3 أحجام: large / medium / compact)
- `BreakingTicker` شريط عاجل متحرك
- `MarketsWidget` بطاقة أسواق (بيانات وهمية في البداية مع إمكانية ربطها بـ API لاحقاً)
- `WeatherWidget` بطاقة طقس
- `SportsWidget` نتائج الدوري
- `SectionTabs` تبويبات للأقسام
- `Masthead` بلوك أسماء الإدارة

## البيانات
- بيانات وهمية (mock) في ملف `src/data/news.ts` بفئات: سياسة، اقتصاد، رياضة، فن، حوادث، تقارير، منوعات
- صور أخبار من Unsplash بمواضيع مناسبة (مصر/سياسة/رياضة/اقتصاد)
- بيانات أسواق ثابتة الآن: EGX30, الذهب, USD/EGP, EUR/EGP, SAR/EGP

## التفاصيل التقنية
- React + Vite + TailwindCSS (متوفر بالفعل)
- `dir="rtl"` و `lang="ar"` على الـ html
- نظام تصميم في `index.css` و `tailwind.config.ts` بـ HSL tokens (primary كحلي، accent ذهبي، breaking أحمر)
- مكونات shadcn للأزرار/التابات/الكاروسيل
- SEO: عنوان + ميتا وصف عربي + H1 واحد + alt للصور + JSON-LD لنوع NewsMediaOrganization
- متجاوب بالكامل (موبايل/تابلت/ديسكتوب)
